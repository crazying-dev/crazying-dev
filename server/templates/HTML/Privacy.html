隐私声明

本站承诺保护你的隐私对我来说不重要

我们收集的信息

本站不主动收集任何个人身份信息。你在浏览本站时，无需注册、无需登录、无需填写任何个人资料。

我们如何使用信息

本站不私自使用任何个人信息。所有内容仅供你在本站的访问都是匿名的。

第三方服务

本站使用了以下第三方服务：

  • Github Pages / Cloudflare Pages / 静态资源托管（访问日志由服务提供商记录）

这些服务可能会记录访问者的 IP 地址等信息，用于统计和安全防护。此类信息由第三方服务提供商管理，本站无法访问或控制。

联系方式

如果你对本声明有任何疑问，请通过 联系我 页面与我取得联系。
